How It Works

Here's a brief overview of how this system was built:

  1. Frontend: Created using HTML, CSS, and JavaScript. The UI allows users to enter their favorite songs and obtain recommendations.
  2. Backend: Built with Node.js and Express.js. It processes the user’s input and fetches recommendations from the Spotify API.
  3. Spotify API: Utilizes the Client Credentials flow to obtain access tokens and retrieve song recommendations based on the user’s favorite tracks.

Total dependencies required:

  • express
  • axios
  • body-parser
