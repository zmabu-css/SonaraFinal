# Music Recommendation System

### Setup

1. **Clone the repository**
    ```sh
    git clone <repository-url>
    ```
2. **Install dependencies**
    ```sh
    cd music-recommendation-system
    npm install
    ```
3. **Run the server**
    ```sh
    node server/app.js
    ```
4. **Open Your Browser**
    - Navigate to `http://localhost:3000`

### APIs Used

- Spotify Web API for music recommendations
